# cwss boilerplate
My Node/Webpack boilerplate V1. 
Contains:
- Express
- Node
- Webpack
- Hot reload
- Babel
- PostCSS
- Recss for sass-like tools
- CssNext
- Dust
- Eslint with Air bnb config
- Scourcemapping enabled
- Compression enabled
- Imgmin
